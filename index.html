⚡ 核心知識
DATA SCALING FUNDAMENTALS · 資料縮放基礎
📏
Min-Max 正規化（Normalization）
線性縮放到 [0, 1]，保留分布形狀。
對離群值高度敏感。
✅ 適用：神經網路、KNN、影像像素。
x' = (x − min) ÷ (max − min)
📊
Z-Score 標準化（Standardization）
均值=0、標準差=1，對離群值較不敏感。
✅ 適用：SVM、PCA、線性回歸。
z = (x − μ) ÷ σ
🛡️
Robust Scaler
中位數 + IQR，對極端值強健。
✅ 適用：含大量離群值的資料集。
x' = (x − Q2) ÷ IQR
⚠️
何時必須縮放？
✅ KNN、SVM、PCA、神經網路
✅ 梯度下降優化算法
❌ 決策樹、隨機森林（基於分裂）
❌ 樸素貝葉斯（獨立假設）
# ── 🔮 阿迦莫多之眼：Scikit-learn 資料縮放 ────────────────── from sklearn.preprocessing import MinMaxScaler, StandardScaler, RobustScaler import numpy as np X = np.array([[38, 1500, 4000], # [年齡, 力量指數, 訓練時數] [17, 7500, 800], [45, 9800, 200]]) # ① Min-Max 正規化 → 值域 [0, 1] mm_scaler = MinMaxScaler() X_mm = mm_scaler.fit_transform(X) # ② Z-Score 標準化 → μ=0, σ=1 std_scaler = StandardScaler() X_std = std_scaler.fit_transform(X) # ③ Robust Scaler → 使用 Q2 + IQR，抗離群值 rob_scaler = RobustScaler() X_rob = rob_scaler.fit_transform(X) # ⚠️ 重要：fit 只在訓練集，transform 同時用於訓練集與測試集！ # scaler.fit(X_train) → scaler.transform(X_test) ← 避免資料洩漏
💡 iPAS 考試重點： 縮放器必須只用 fit_transform() 於訓練集，測試集只能用 transform()，否則會造成資料洩漏（Data Leakage），讓測試集資訊污染了模型訓練！

⚡ 核心知識
OUTLIER DETECTION METHODS · 異常值偵測方法
📦
IQR 四分位距法
IQR = Q3 − Q1
下界 = Q1 − 1.5 × IQR
上界 = Q3 + 1.5 × IQR
超出邊界 → 判定異常
x < Q1-1.5IQR 或 x > Q3+1.5IQR
🔢
Z-Score 方法
計算標準分數，|z| > 3 判定異常。
假設資料近似常態分布，
99.7% 數據落在 ±3σ 內。
|z| = |x − μ| / σ > 3 → 異常
🤖
Isolation Forest
無監督方法，透過隨機分割，
異常點更容易被孤立（路徑短）。
適用高維度複雜資料。
路徑越短 → 異常分數越高
🎯
處理策略
🗑️ 刪除：確認為錯誤記錄
📦 封頂：Winsorization（截尾）
🔄 替換：中位數/均值填充
🏷️ 標記：作為特殊類別保留
# ── 🦸 魔浮斗篷：異常值偵測與處理 ────────────────────────── import numpy as np import pandas as pd from scipy import stats from sklearn.ensemble import IsolationForest scores = np.array([45,52,48,61,55,999,43,58,47,-200,50]) # ① IQR 法 Q1, Q3 = np.percentile(scores, [25, 75]) IQR = Q3 - Q1 lower, upper = Q1 - 1.5*IQR, Q3 + 1.5*IQR iqr_mask = (scores < lower) | (scores > upper) # True = 異常 # ② Z-Score 法 z_scores = np.abs(stats.zscore(scores)) z_mask = z_scores > 3 # |z| > 3 # ③ Isolation Forest（適合高維） iso = IsolationForest(contamination=0.1, random_state=42) labels = iso.fit_predict(scores.reshape(-1,1)) # -1=異常, 1=正常 # ④ Winsorization（截尾處理） cleaned = np.clip(scores, lower, upper) # 超界值截斷
💡 iPAS 考試重點： IQR 法不假設分布、適合偏態資料；Z-Score 法假設常態分布；Isolation Forest 適合高維度無監督偵測。 contamination 參數為預估異常比例（0到0.5）。

⚡ 核心知識
PRINCIPAL COMPONENT ANALYSIS · 主成分分析
🌀
PCA 是什麼？
無監督降維技術，將高維資料投影到低維空間，最大化保留方差（資訊量）。新軸稱為「主成分（Principal Component）」。
📐
PCA 運作步驟
① 標準化資料（必要！）
② 計算協方差矩陣
③ 計算特徵值與特徵向量
④ 選取前 k 個主成分
⑤ 投影資料到新空間
📈
解釋方差比例 EVR
每個主成分解釋一定比例的總方差。常見做法：保留累積解釋方差達 80–95% 的成分數。Scree Plot 視覺化選擇。
EVR = λᵢ / Σλ (λ為特徵值)
⚠️
PCA 注意事項
✅ 必須先標準化！
✅ 主成分是原特徵的線性組合
❌ 失去可解釋性
❌ 僅適用線性關係（非線性用 t-SNE, UMAP）
❌ 分類問題考慮 LDA 替代
# ── 💍 懸戒：PCA 降維完整流程 ─────────────────────────────── from sklearn.preprocessing import StandardScaler from sklearn.decomposition import PCA import numpy as np import matplotlib.pyplot as plt # 假設資料有 100 個特徵 X = np.random.randn(1000, 100) # 1000 樣本, 100 特徵 # ① 必須先標準化！（PCA 對尺度敏感） scaler = StandardScaler() X_std = scaler.fit_transform(X) # ② 初步 PCA（保留全部成分，查看解釋方差） pca_full = PCA() pca_full.fit(X_std) evr = pca_full.explained_variance_ratio_ # 每個PC的解釋比例 cum_evr = np.cumsum(evr) # 累積解釋比例 # ③ 選擇保留 95% 方差的成分數 n_comp = np.argmax(cum_evr >= 0.95) + 1 print(f"保留 95% 方差需要 {n_comp} 個主成分") # ④ 執行降維 pca = PCA(n_components=n_comp) X_pca = pca.fit_transform(X_std) print(f"降維：{X.shape[1]}維 → {X_pca.shape[1]}維") # ⑤ Scree Plot（手肘圖） plt.plot(np.cumsum(evr[:20])) plt.axhline(y=0.95, color='r', linestyle='--', label='95% threshold') plt.xlabel('Number of Components'); plt.ylabel('Cumulative EVR')
💡 iPAS 考試重點： PCA 前必須標準化，否則大尺度特徵主導主成分方向。用 explained_variance_ratio_ 查看每個 PC 貢獻，用 n_components=0.95 可直接指定保留 95% 方差的成分數。
